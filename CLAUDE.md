## 🎨 코딩 스타일
✅ Complete 프로젝트는 공부를 완료한 코드들이다. 내 스타일에 맞게 수정된 코드들 이기 때문에, 코드 스타일을 참고한다.

### 📦 헤더 및 네임스페이스
```cpp
#include <bits/stdc++.h>
using namespace std;
```
- **모든 라이브러리**: `bits/stdc++.h`로 통합 (코테 최적화)
- **네임스페이스**: `using namespace std` 사용
- **성능보다 속도**: 빠른 코딩을 위한 선택

### 🌐 전역 변수 선언
```cpp
// 전역 변수들은 상단에 선언하고 정렬
int    visited[54][54], a[54][54], n, l, r;
int    sum = 0, cnt = 0;
bool   flag = false;
vector<pair<int,int>> uni;
```

**정렬 규칙:**

✅ **타입 정렬**: 동일한 타입끼리 그룹화

✅ **컬럼 정렬**: 타입 / 변수명 / = / 초기값 정렬

✅ **의미별 그룹**: 관련 변수들끼리 묶어서 선언

### 💬 주석 스타일
```cpp
// 변수 설명 주석
int visited[54][54], a[54][54], n, l, r, sum, cnt;
// visited: 방문 체크 배열 (54x54로 여유있게 선언)
// a: 각 나라의 인구수를 저장하는 배열
// n: 땅의 크기, l: 최소 인구 차이, r: 최대 인구 차이

// 라인별 주석 정렬
visited[ny][nx] = 1;                   // 방문 처리
uni.push_back({ny, nx});               // 연합에 추가
sum += a[ny][nx];                      // 연합의 총 인구수에 추가
dfs(ny, nx);                           // 재귀적으로 DFS 계속 진행
```

**주석 규칙:**

✅ **변수 설명**: 전역 변수 선언 후 각각의 역할 설명

✅ **함수 설명**: 매개변수와 동작 방식 설명

✅ **라인 정렬**: 같은 블록 내 주석들의 시작 위치 통일

✅ **명확성**: 알고리즘 로직을 이해하기 쉽게 설명

## 🌏 한글 인코딩 설정

### 🔧 UTF-8 BOM 설정 (한글 주석 지원)

**⚠️ 중요 (Important):** Windows 환경에서 한글 주석이 깨지지 않게 하려면 반드시 UTF-8 BOM을 사용해야 합니다.

#### 📋 Claude Code에서 파일 생성 시 설정
```bash
# UTF-8 BOM을 포함한 파일 생성 방법
echo -e "\xEF\xBB\xBF#include <bits/stdc++.h>" > filename.cpp
```

**🔍 BOM 바이트 구조:**
- `\xEF\xBB\xBF` = UTF-8 BOM (Byte Order Mark)
- 파일 맨 앞 3바이트로 UTF-8 인코딩임을 시스템에 알림

#### 🛠️ Visual Studio 수동 설정 방법

**방법 1: 파일별 인코딩 설정**
1. 🔸 Visual Studio에서 .cpp 파일 열기
2. 🔸 `파일` → `고급 저장 옵션` (Advanced Save Options)
3. 🔸 인코딩을 `유니코드(UTF-8 서명 있음) - 코드 페이지 65001`로 변경
4. 🔸 저장 후 한글 주석이 제대로 표시되는지 확인

**방법 2: 프로젝트 전체 설정**
1. 🔸 프로젝트 속성 → `구성 속성` → `C/C++` → `명령줄`
2. 🔸 추가 옵션에 `/utf-8` 추가
3. 🔸 모든 .cpp 파일에 적용됨

**방법 3: 메모장 활용**
1. 🔸 메모장에서 코드 작성
2. 🔸 저장 시 인코딩을 `UTF-8`로 선택
3. 🔸 Visual Studio에서 해당 파일 열기

#### 🎯 Claude Code 사용 시 권장 사항
